## 🛠️ 安装与配置

### 第一步：安装 Alma

访问 [Alma 官网](https://alma.now/) 下载并安装适合你操作系统的版本。

### 第二步：设置中文界面

为了更好的使用体验，建议将界面设置为中文：

1. 打开 Alma 应用
2. 进入 **设置 → 通用 → 语言**
3. 选择「中文」

![设置中文界面](/images/apps/alma/image.png)

### 第三步：添加 IkunCode 自定义供应商

**1. 进入供应商设置**

点击 **选择供应商 → Add Custom Provider**（添加自定义提供商）

![添加供应商](/images/apps/alma/image%201.png)

**2. 配置 IkunCode 供应商**

填写以下信息：
- **Provider Name**：IkunCode（或自定义名称）
- **Base URL**：`https://api.ikuncode.cc/v1`
- **API Key**：粘贴你从 [IkunCode 控制台](https://api.ikuncode.cc/console/token) 创建的 API Key

::: tip 💡 提示
- Claude 模型：确保使用 C开头的背带裤分组的 API Key
- Gemini 模型：确保使用 gemini分组的 API Key
- IkunCode 的 Codex 暂不支持第三方客户端调用
:::

**3. 获取可用模型**

添加完成后，在 IkunCode 供应商界面点击 **Fetch**（获取）按钮，系统会自动获取所有可用模型。

![Fetch 获取模型](/images/apps/alma/image%202.png)

选择你需要的模型即可开始使用。

### 第四步：配置多个分组（可选）

IkunCode 支持通过不同的 API Key（不同分组）访问不同的模型集合。

在供应商界面，你可以：
1. 填写不同分组的 API Key
2. 获取对应分组的模型列表
3. 根据需求切换使用

![不同分组配置](/images/apps/alma/image%205.png)

![模型选择](/images/apps/alma/image%206.png)

::: tip 💡 提示
如需使用 Gemini 3 的画图功能，需要选择 **default** 渠道的 API Key。
:::

## 💬 新建聊天与使用

### 重要设置提醒

在新建聊天时，**上下文范围** 设置非常重要：

![上下文设置](/images/apps/alma/image%203.png)

::: warning ⚠️ 强烈建议
**选择「全选」而非「自动」！**

经测试，选择「自动」时模型会产生严重的幻觉问题，可能导致：
- 引用不存在的文件
- 生成错误的代码逻辑
- 误解项目结构

选择「全选」可以确保模型获得完整的上下文信息，提供更准确的响应。
:::

## 常见问题

### 如何切换不同的 API 提供商？

在聊天界面顶部的模型选择器中，可以随时切换不同的提供商和模型。

### 为什么有些模型无法使用？

- 检查对应的 API Key 是否有权限访问该模型
- 确认账户余额是否充足
- 查看 [IkunCode 控制台](https://api.ikuncode.cc/console/token) 的令牌组设置（Claude 用 C开头的背带裤分组，Gemini 用 gemini分组）

### Alma 和 Claude Code 有什么区别？

| 特性 | Alma | Claude Code |
|------|------|-------------|
| 界面 | 图形化界面 | 命令行界面 |
| 适用场景 | 全功能开发环境 | 终端快速操作 |
| 集成功能 | 终端、Git、浏览器等 | 纯命令行工具 |
| 学习曲线 | 较低 | 较高 |

可以根据具体需求选择使用。
