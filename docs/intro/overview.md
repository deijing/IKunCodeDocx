## How It Works

```
Your machine ←→ IKunCode relay ←→ AI service provider
              (local access)      (overseas service)
```

IKunCode sits in the middle and:
1. Receives your request
2. Forwards it to the AI service provider
3. Returns the response to you

The whole process is transparent — the experience is identical to using the official service.
